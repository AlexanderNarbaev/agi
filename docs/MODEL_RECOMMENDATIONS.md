# Рекомендации моделей для MATRIX pretraining

**Дата:** 2026-07-06
**Статус:** Актуально

---

## Контекст

MATRIX конвертирует веса трансформеров в таблицы истинности нейронов (Avro формат). Процесс:
1. Загрузка весов из HuggingFace (safetensors)
2. Извлечение матриц внимания и FFN слоёв
3. Квантизация в булевы таблицы истинности (k входов → 2^k выходов)
4. Сохранение в Avro формат

### Текущие интеграции

| Модель | Параметры | Нейроны | Слои | k | Размер Avro |
|--------|-----------|---------|------|---|-------------|
| SmolLM2-135M | 135M | 180 | 6 | 12 | ~370 KB |
| Qwen2.5-0.5B | 500M | 720 | 24 | 16 | ~5.9 MB |

## Рекомендации по интеграции

### Немедленная интеграция (совместимый пайплайн)

1. **Qwen3-1.7B** — то же семейство Qwen, 28 слоёв, thinking mode. Drop-in upgrade от Qwen2.5-0.5B.
2. **Qwen2.5-1.5B** — прямое 3× масштабирование существующего 0.5B. Нулевые изменения пайплайна.
3. **Qwen3-0.6B** — быстрая песочница с thinking mode.

### Максимальное качество рассуждений

1. **DeepSeek-R1-Distill-Qwen-1.5B** — R1 дистилляция = самое глубокое рассуждение на 1.5B. MIT.
2. **Phi-4-mini-instruct** — лучшие математические результаты на ~4B. 32 слоя = самое глубокое извлечение.

### Архитектурное разнообразие

1. **SmolLM3-3B** — уникальная архитектура GQA+NoPE 3:1. Другие паттерны таблиц истинности.
2. **Gemma 3 1B** — архитектура Google. Мультимодальные веса могут кодировать визуальные паттерны.

### Для поведения Minecraft-бота

Извлечённые веса управляют принятием решений ботом. Модели с **сильным coding + reasoning + agent capabilities** дадут лучшее поведение:

1. **Qwen3-1.7B** — нативный agent/tool-calling, thinking mode для сложных решений
2. **DeepSeek-R1-Distill-Qwen-1.5B** — глубокие цепочки рассуждений для планирования
3. **SmolLM2-360M** — быстрая итерация, доказано в пайплайне

## План действий

### Phase 1: Быстрые победы (1-2 дня)

```bash
# 1. Скачать Qwen3-0.6B (самая маленькая, быстрая итерация)
huggingface-cli download Qwen/Qwen3-0.6B --local-dir models/Qwen3-0.6B

# 2. Конвертировать
python3 scripts/pretrain_neurons.py \
  --model-path models/Qwen3-0.6B \
  --output-dir models/pretrained/qwen3-0.6b \
  --layers 6 --neurons-per-layer 30

# 3. Протестировать интеграцию
./gradlew :matrix-core:test
```

### Phase 2: Основная модель (2-3 дня)

```bash
# 1. Скачать Qwen3-1.7B
huggingface-cli download Qwen/Qwen3-1.7B --local-dir models/Qwen3-1.7B

# 2. Конвертировать (все 28 слоёв)
python3 scripts/pretrain_neurons.py \
  --model-path models/Qwen3-1.7B \
  --output-dir models/pretrained/qwen3-1.7b \
  --layers 6 --neurons-per-layer 30

# 3. Обновить AgentBrainService для приоритизации Qwen3
# 4. Пересобрать Docker image и деплой в minikube
```

### Phase 3: Разнообразие (3-5 дней)

```bash
# 1. Скачать DeepSeek-R1-Distill-Qwen-1.5B (MIT, лучший reasoning)
huggingface-cli download deepseek-ai/DeepSeek-R1-Distill-Qwen-1.5B \
  --local-dir models/DeepSeek-R1-Distill-Qwen-1.5B

# 2. Конвертировать и интегрировать
# 3. A/B тестирование: Qwen3 vs DeepSeek-R1 в Minecraft
```
