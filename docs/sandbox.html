About MPDT Neurons

MPDT (McCulloch-Pitts Decision Tree) neurons are discrete logic cells. Unlike neural networks, they use no floating-point numbers — only Boolean logic. Each output is a deterministic table lookup (nanoseconds). Training uses genetic algorithms, not gradient descent.

No hallucinations. Full interpretability. Nanosecond inference.
