Что такое MATRIX?

MATRIX — это система искусственного интеллекта нового поколения. В отличие от больших языковых моделей, которые галлюцинируют и требуют гигаватты энергии, MATRIX использует дискретные логические нейроны (MPDT).


Каждый нейрон — это интерпретируемая булева функция, которая обучается не градиентным спуском, а эволюционным алгоритмом. Система спроектирована так, чтобы быть энергоэффективной, децентрализованной и этически подотчётной. В её ядро встроены неотменяемые этические ограничения — Три запрета (не убий, не пытай, не порабощай).



Ключевые преимущества

Не галлюцинирует

Логический вывод детерминирован и верифицируем. Каждое решение — цепочка булевых операций, которую можно проверить.

Интерпретируема

Любое решение объяснимо как дерево логических правил. Никаких «чёрных ящиков».

Энергоэффективна

Инференс на таблице истинности — наносекунды. Работает на микроконтроллерах (ESP32). На порядки экономичнее нейросетей.

Этична по конструкции

FROZEN-нейроны блокируют нарушение Трёх запретов. Система имеет право отказаться от преступного приказа (ЭЛЕВТЕРИЯ).

Децентрализована

Каждый может запустить свой инстанс. Знания распространяются через Ноосферу — глобальное хранилище FNL.

Открыта

AGPLv3 + этические ограничения. Весь код, все спецификации, все решения — публичны.



Как это работает

В основе MATRIX лежит MPDT-нейрон (McCulloch-Pitts Decision Tree) — дискретная логическая ячейка, хранящая таблицу истинности для K входов. Нейроны объединяются в кластеры, кластеры — в функциональные доли (FNL). Обучение ведётся генетическим алгоритмом: популяция нейронов эволюционирует через мутации, скрещивание и отбор по фитнес-функции.


Иерархия Медиаторов управляет драйверами поведения (Energy, Curiosity, Safety), обеспечивая проактивность. Этический фильтр проверяет каждое действие на соответствие Трём запретам. Распределённая архитектура на Apache Pekko + Kafka + MinIO обеспечивает масштабируемость и отказоустойчивость.


Стек: Java 25, Quarkus 3.35.4, Apache Pekko 1.6.0, Apache Kafka, MinIO, Avro, JaCoCo, Prometheus, Grafana, Jaeger.



Быстрый старт

git clone https://gitverse.ru/AlexandrNarbaev/agi.git
cd agi
./gradlew build
./gradlew test                         # 414 тестов
./gradlew :matrix-core:quarkusBuild -Dquarkus.package.jar.type=uber-jar
java -jar matrix-core/build/matrix-core-*-runner.jar demo

Ожидаемый результат: ALL SYSTEMS NOMINAL — все 8 фаз проходят проверку.


Запуск GridWorld-симуляции (эволюция агента):

java -jar matrix-core/build/matrix-core-*-runner.jar simulate -g 20 -p 10 -k 16 --seed 42
