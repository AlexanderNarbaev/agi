## 1. Architecture Overview

GridWorld simulator with partial observability + evolutionary training of an agent
with 4 MPDT neuron specialists.

## 2. Domain Model (DDD)

### Bounded Context: `simulation`

**Entities:**
- `AgentBody` — mutable position, energy during simulation. Identity: id.
- `World` — mutable grid state. Identity: seed/hash.

**Value Objects:**
- `Position(int x, int y)` — immutable
- `CellType` — enum

**Aggregates:**
- `World` (root: `Grid` cells) — encapsulates all world state
- `Agent` (root: `AgentBody`, has `AgentBrain`) — brain is immutable during tick

**Domain Events:**
- `AgentMoved(from, to, direction)`
- `ResourceCollected(type, position)`
- `WallCollision(position, direction)`
- `EnergyDepleted(agentId)`
- `TickCompleted(tickNumber)`

### Bounded Context: `evolution`

**Entities:**
- `Population` — mutable collection of Chromosomes

**Value Objects:**
- `FitnessScore(long raw)` — food collected minus penalties

**Aggregates:**
- `Population` (root: list of Chromosome)

**Domain Events:**
- `GenerationCompleted(generation, bestFitness, avgFitness)`
- `MutationApplied(chromosomeId, operator)`

## 3. Sensor Model

Agent sees Moore neighborhood (3x3, 8 neighbors). K_MAX raised to 20.

```
Bits 0-1:   N  cell (00=empty, 01=wall, 10=resource, 11=agent)
Bits 2-3:   NE cell
Bits 4-5:   E  cell
Bits 6-7:   SE cell
Bits 8-9:   S  cell
Bits 10-11: SW cell
Bits 12-13: W  cell
Bits 14-15: NW cell
Bits 16-17: Energy level (00=critical, 01=low, 10=med, 11=high)
```

Total: 18 bits. K_MAX = 20 (absolute max per L1).

---

## 4. Agent Brain

4 MPDT neurons, each specialized for one direction:
- `N_NEURON`: should I go North?
- `S_NEURON`: should I go South?
- `W_NEURON`: should I go West?
- `E_NEURON`: should I go East?

Each neuron: k=18 DecisionTree → 1 bit output.
Action selection: argmax of 4 outputs. If tie or all false → STAY (conserve energy).

Each neuron is independently evolvable (separate Chromosome).

---

## 5. Genetic Algorithm

Per L5 §2.3:
- Population: P = 20
- Generations: G = 50
- Per agent: 4 chromosomes (one per direction)
- Selection: tournament (size 3)
- Elitism: keep best P/2
- Mutation operators: all 8 from GeneticOperators
- Fitness: avg score over 5 simulation runs

---

## 6. World Rules

- Grid: 20×20, toroidal (wrap-around)
- Resources: 10 food items, respawn when collected (random empty cell)
- Obstacles: 15 walls, static
- Agent: starts at (0,0), energy = 100
- Each move: -1 energy
- Collect food: +20 energy
- Hit wall: -5 energy, don't move
- Energy = 0: simulation ends
- Max ticks: 200
