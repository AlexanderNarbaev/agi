## 1. Architecture Overview

Single-node Pekko Typed Actor for batched MPDT neuron inference with in-memory event journal.

### C4 Level 2

```
┌─ NeuronClusterActor (Pekko Typed) ─────────────────────┐
│  Port: neuron cluster commands & signals                │
│  Adapters: Kafka (later), REST (later)                  │
│  Domain:                                                │
│    activeNeurons: Map<NeuronId, NeuronInstance>          │
│    SignalBuffer: RingBuffer                              │
│    EventLog: List<NeuronEvent>                          │
│                                                         │
│  Inference Cycle (T_cycle = 1ms):                       │
│  1. drain() signal buffer                               │
│  2. evaluateBatch() — direct long[] LUT lookup          │
│  3. emitSignals() — place changed outputs in buffer     │
└─────────────────────────────────────────────────────────┘
```

### C4 Level 3 — Components

| Component | Class | Responsibility |
|-----------|-------|---------------|
| NeuronInstance | cluster/NeuronInstance | Wraps TruthTable with state, lastOutput, mutable flag |
| Signal | cluster/Signal | Record: sourceId, targetId, value, timestamp |
| SignalBuffer | cluster/SignalBuffer | Ring buffer for incoming/outgoing signals |
| ClusterConfig | cluster/ClusterConfig | Configuration: T_cycle, maxNeurons, batchSize |
| NeuronClusterActor | cluster/NeuronClusterActor | Pekko Typed Actor: commands, inference loop |
| NeuronEvent | cluster/events/NeuronEvent | Avro event: CREATED, MUTATED, FROZEN, REMOVED |
| NeuronClusterSerializer | cluster/serialization/ | Jackson+Avro serialization for Pekko |

### Commands

```java
sealed interface Command {
    record LoadNeuron(TruthTable table) {}
    record UnloadNeuron(NeuronId id) {}
    record FreezeNeuron(NeuronId id) {}
    record InjectSignal(Signal signal) {}
    record EvaluateTick() {}
    record GetMetrics() {}
    record GetNeuronCount() {}
}
```

## 5. Exclusions (for Phase 1.1)
- No Pekko Cluster (single node)
- No Kafka (in-memory signal injection)
- No RocksDB (in-memory state only)
- No Mediators
- No .ldn snapshots
