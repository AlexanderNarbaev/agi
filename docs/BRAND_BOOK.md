# MATRIX Brand Book v1.0

## Logo

The MATRIX logo represents the core concepts of the system:
- **Hexagons** — network structure, neuroclusters, interconnectedness
- **Decision tree nodes** — MPDT neurons, binary logic, input→decision→output
- **Spiral** — evolution, growth, DNA, the "Spiral Canon" ethics
- **Inner grid** — Truth Table (2^k boolean outputs), discrete computation
- **Gradient colors** — cyan (#00d4ff) for logic/truth, purple (#7b2ff7) for depth/complexity

- **Headings:** JetBrains Mono (monospace) — reflects code, logic, precision
- **Body:** Inter / system-ui sans-serif
- **Code:** JetBrains Mono / Fira Code

### Key Messages

1. **"Discrete logic, not statistics."** — MPDT neurons don't use floats or gradients.
2. **"No hallucinations."** — Deterministic boolean output.
3. **"Interpretable by design."** — Every decision is a readable boolean formula.
4. **"Evolves without gradients."** — Genetic algorithm, not backpropagation.
5. **"Ethics are axioms, not afterthoughts."** — FROZEN ethical filter at the core.
6. **"Nanosecond inference."** — Table lookup, faster than any matrix multiply.

### Voice & Tone

- **Technical but accessible** — explain concepts, don't intimidate
- **Honest** — no hype, no "AGI killer", no clickbait
- **Curious** — open questions, invitations to collaborate
- **Ethical** — always contextualize within the Spiral Canon
