# МАТРИЦА (MATRIX)

Модульная Архитектура Трансцендентных Рекурсивных Интеллектуальных Цепочечных Автоматов.

## Статус

Фаза 0 «Искра» — реализация базового MPDT-нейрона и генетического алгоритма.

## Документация

Все спецификации находятся в папке `docs/`:

| Документ | Содержание |
|----------|------------|
| L0_manifesto.md | Манифест, аксиомы, миссия |
| L1_MPDT_neuron.md | Формальная спецификация MPDT-нейрона |
| L2_Iteraction_protocol.md | Протоколы взаимодействия, консенсус |
| L3_Neurocluster_Arch.md | Архитектура нейронных кластеров и FNL |
| L4_Mediator.md | Медиатор, драйверы, проактивность |
| L5_DNA.md | Эволюционное обучение, сжатие, безопасность |
| L6_Memory.md | Хранилище, снапшоты, Ноосфера |
| L7_Ethics.md | Мультимодальный прокси, Этический фильтр |
| L8_Roadmap.md | Дорожная карта и протокол разработки |

## Быстрый старт

1. Установите JDK 25, Gradle 8+.
2. Клонируйте репозиторий.
3. Запустите сборку: `./gradlew build`
4. Запустите тесты: `./gradlew test`

## Разработка с OpenCode
